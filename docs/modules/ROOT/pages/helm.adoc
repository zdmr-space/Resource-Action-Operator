== Install The Operator From The Published Helm Repository

[source,bash]
----
set -euo pipefail

RELEASE_TAG="$(curl -fsSL https://api.github.com/repos/zdmr-space/Resource-Action-Operator/releases/latest \
  | sed -n 's/.*"tag_name":[[:space:]]*"\([^"]*\)".*/\1/p' \
  | head -n1)"

if [ -z "${RELEASE_TAG}" ]; then
  echo "Could not determine latest GitHub release tag" >&2
  exit 1
fi

CHART_VERSION="${RELEASE_TAG#v}"

helm repo add resource-action-operator https://zdmr-space.github.io/Resource-Action-Operator
helm repo update

helm upgrade --install resource-action-operator resource-action-operator/resource-action-operator \
  --version "${CHART_VERSION}" \
  --namespace resource-action-operator-system \
  --create-namespace
----

Notes:

- The operator chart contains the CRD in `charts/resource-action-operator/crds/`.
- Helm installs the CRD during chart install.
- Helm does not remove the CRD automatically on uninstall.

== Install A Simple HTTP ResourceAction With Helm

This example installs a generic `ResourceAction` that watches created `Namespace` objects and sends an HTTP request.

[source,bash]
----
helm upgrade --install namespace-http resource-action-operator/resource-action \
  --version "${CHART_VERSION}" \
  --namespace default \
  -f - <<'EOF'
selector:
  group: ""
  version: v1
  kind: Namespace

events:
  - Create

action:
  type: http
  mode: once

http:
  method: POST
  url: https://example.internal/hooks/namespaces
  expectedStatus: "^2..$"
  body:
    template: |
      {
        "event": "namespace-create",
        "name": "{{ .metadata.name }}"
      }
EOF
----

== Local Chart Installs

Install the operator from the local checkout:

[source,bash]
----
helm upgrade --install resource-action-operator charts/resource-action-operator \
  --namespace resource-action-operator-system \
  --create-namespace
----

Enable the admission webhook with cert-manager:

[source,bash]
----
helm upgrade --install resource-action-operator charts/resource-action-operator \
  --namespace resource-action-operator-system \
  --create-namespace \
  --set webhook.enabled=true \
  --set webhook.certManager.enabled=true
----

Install the generic action chart from the local checkout:

[source,bash]
----
helm upgrade --install generic-action charts/resource-action \
  --namespace default
----

Install the job-focused chart from the local checkout:

[source,bash]
----
helm upgrade --install deployment-job charts/resource-action-job \
  --namespace default
----
